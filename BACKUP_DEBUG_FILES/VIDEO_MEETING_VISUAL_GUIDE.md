## 🔍 STEP-BY-STEP GUIDE

### Step 1: Navigate to Sessions
```
Sidebar → Click "Sessions" (2nd item, calendar icon)
```

### Step 2: Find Your Session
```
Look for your scheduled session in the list
Each session is displayed as a card
```

### Step 3: Locate Meeting Section
```
Inside each session card, scroll down
You'll see a colored box (blue or green)
This is the meeting link section
```

### Step 4: Generate Meeting Link
```
Click the "Start Meeting Now" button
Wait 2-3 seconds for generation
Button changes to "Join Video Session"
```

### Step 5: Join Meeting
```
Click "Join Video Session"
Opens Jitsi in new tab
Both student and counsellor see same link
```

**Sidebar → Sessions → Inside each session card**

## 🔍 TROUBLESHOOTING

### "I don't see any meeting section"
✅ **Solution**: 
1. Make sure you're on the **Sessions page** (not Dashboard)
2. Look **inside** the session card (scroll down if needed)
3. Check that the session is **scheduled** (not completed/cancelled)

### "I only see a countdown"
✅ **Solution**: 
- Click the **"Start Meeting Now"** button
- Don't wait for the countdown to finish
- Meeting will generate immediately

### "Button says 'Join Video Session' but nothing happens"
✅ **Solution**: 
- Check if pop-ups are blocked
- Try clicking again
- Refresh the page and try again

### "I'm a student and don't see the meeting link"
✅ **Solution**: 
- Go to `/student/sessions` (not `/student`)
- Look for **scheduled** sessions
- Meeting section is at the bottom of each session card

### "I'm a counsellor and don't see the meeting link"
✅ **Solution**: 
- Go to `/counsellor/sessions` (not `/counsellor`)
- Each session is now a **separate card**
- Meeting section is at the bottom of each card
